js脚本

window.onload = function(){

//获取所需要操作的对象
var smallBox = document.getElementById(“small-box”);
var floatBox = document.getElementById(“float-box”);
var bigBox = document.getElementById(“big-box”);
var bigImage = bigBox.getElmentsByTagName(“img”)[0];

//鼠标移动到对象上显示放大镜和放大镜放大后的图像
smallBox.onmouseover = function(){
floatBox.style.display = “block”;
bigBox.style.display = “block”;
}

//鼠标移出对象时隐藏放大镜和放大镜放大后的图像
smallBox.onmouseout = function(){
floatBox.style.display = “none”;
bigBox.style.display = “none”;
}

//鼠标在对象上移动触法的事件
smallBox.onmousemove = function(e){
var event = e || window.event;
var left = event.clientX - smallBox.offsetLeft - floatBox.offsetWidth/2;
var top = event.clientY - smallBox.offsetTop - floatBox.offsetHeight/2;

if(left<0){
    left = 0;
}else if (left > (smallBox.offsetWidth - floatBox.offsetWidth)){
left = smallBox.offsetWidth - floatBox.offsetWidth;
}

if(top<0){
    top = 0;
}else if (top > (smallBox.offsetHeight - floatBox.offsetHeight)){
top = smallBox.offsetHeight - floatBox.offsetHeight;
}

bigImage.style.left = -left/smallBox.offsetWidth*bigImage.offsetWidth +"px";
bigImage.style.top = -top/smallBox.offsetHeight*bigImage.offsetHeight+"px";

}

}

第二篇

终于搭建完成了，原来之前是将github仓库误认为是访问地址，现在知道是访问“http://checkfrank.github.io/”（根据自己的github帐户名修改）
这个网址可在仓库的settings->Github Pages看到
搭建过程详见http://blog.sina.com.cn/s/blog_4c44643f0102vuju.html
hexo文档：https://hexo.io/zh-cn/docs/setup.html

总结一下大致过程：

安装Hexo

1.任意位置Git Bash，输入命令“npm install -g hexo
2.创建Hexo文件夹后，进入，Git Bash输入“hexo init”
3.安装依赖包：“npm install”
4.执行“hexo generate”和“hexo server”即可在本地localhost：4000查看

创建Repository

注意名字是Yourname.github.io
1.修改配置文件_config.yml里的deploy，
“type: git
repo: https://github.com/becky76/becky76.github.io
branch: master”

设置SSH keys

配置SSH的官方教程是：https://help.github.com/articles/generating-ssh-keys/
1.先检查是否存在了SSH keys “ls -al ~/.ssh
2.存在的话直接删除.ssh文件夹里面所有文件，.ssh文件夹在密钥文件在C:\Users\n.ssh
3.输入以下指令生成新的ssh key“ssh-keygen -t rsa -C “xxx@163.com””（邮箱是注册Github时用的邮箱），提示输入passphrase时可以直接回车不输入。
4.启动ssh-agent “ssh-agent -s”
5.添加新ssh key到ssh-agent “ssh-add ~/.ssh/id_rsa”
6.成功后拷贝id_rsa.pub到settings->SSH keys添加ssh key
7.是否成功添加SSH key可通过“ssh -T git@github.com”查询

完成部署

1.“hexo generate”和“hexo deploy”，浏览器打开“http://becky76.github.io/”

写文章

1.hexo new “title”
2.编辑source_posts\article.md
注意每次修改本地文件后，需要在C：\Hexo目录下键入“hexo generate”才能保存
